Fix ‚ÄúNo such file or directory‚Äù in g++

  1. Avoid spaces in filenames (or quote them).
  2. Use: g++ -std=c++20 data_types.cpp -o data_types
  3. Run with: ./data_types (or ./data_types.exe on Windows).

Export SVG for vector logos

  1. Remove background (PNG with transparency).
  2. Trace in vector tool; export as .svg.
  3. Embed SVG inline for crisp scaling on web.
